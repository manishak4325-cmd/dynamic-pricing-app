# 💰 Dynamic Pricing System

🚀 A real-time web application that dynamically adjusts prices based on **demand, supply, and time factors** — inspired by systems like Uber surge pricing and airline ticket pricing.

---

## ✨ Features

* 📈 Demand & supply-based pricing
* ⏰ Peak-hour surge pricing
* 📦 Category-based adjustments (Cab, Flight, Hotel)
* 📊 Interactive Demand vs Price graph
* 🌐 Deployed web app using Streamlit

---

## 🧠 Algorithm Used

* **Greedy Optimization Approach**
* Computes optimal price based on current inputs in real-time

---

## 🛠️ Tech Stack

* Python 🐍
* Streamlit 🌐
* Matplotlib 📊
